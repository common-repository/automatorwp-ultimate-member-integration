Connect AutomatorWP with Ultimate Member

== Description ==

> **Important:** Now all free integrations are included in [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") so you no longer need to install them one by one!
>
> We will continue working to improve AutomatorWP and make it easier, faster and more accessible to everyone.

[Ultimate Member](https://wordpress.org/plugins/ultimate-member/ "Ultimate Member") helps you build any kind of community website using WordPress, with member profiles, activity streams, user groups, messaging, and more.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with Ultimate Member adding new triggers and actions that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User account gets approved.
* User account gets marked as inactive.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/ultimate-member/ "AutomatorWP - Ultimate Member") available that let's you expand the integration between AutomatorWP and Ultimate Member bringing you more ways to automatize your community.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
